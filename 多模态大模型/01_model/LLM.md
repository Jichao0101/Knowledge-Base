
# 1 基础机制

大语言模型是一个自回归条件概率模型：

$P(x_t | x_{<t})$

模型在每一步预测下一个 token。

---

# 2 预训练目标

预训练阶段通过最大化似然函数：$\max \log P(x)$，等价于最小化Cross Entropy

优化目标：提高训练语料中真实 token 的预测概率。

因此预训练模型本质是语言分布拟合器, 而不是显式任务求解。

---

# 3 能力来源

语言模型能力主要来自三个因素：

## 3.1 模型容量

更大的模型可以表示更复杂的分布。

## 3.2 数据规模

更大语料覆盖更多模式。

## 3.3 上下文长度

更长上下文提供更多条件信息。

---

# 4 后训练：优化信号来源

后训练改变模型分布。

优化信号来源分为两类：

1 监督信号  
2 奖励信号

---

## 4.1 监督微调（SFT）

信号来源：专家给定答案。

目标函数：$min CE(y_{expert}, y_{model})$

作用：

- 学习对话格式
- 学习任务结构

---

## 4.2 奖励优化

当任务没有明确答案时：使用奖励模型。

常见方法：

RLHF  
DPO

目标形式：

$max E[R(x)] − \beta KL(π || π_{ref})$

---

# 5 系统条件扩展

语言模型可以写为：

$P(x_t | x_{<t}, C)$

其中 C 为系统条件。

C 可以包含：

- system prompt
- 历史对话
- 检索信息
- 工具结果

---

# 6 幻觉机制

当：

context 信息不足  
或训练分布存在虚构模式

模型仍会输出概率最高 token。

因此：幻觉是统计一致性与事实一致性不对齐的结果。

---

# 7 幻觉控制方法

1 修改 reward  
2 扩展条件信息  
3 引入外部验证

例如：

- RAG
- Tool execution
- Code execution

---

# 8 局限

语言模型存在三个结构限制：

1 无环境状态  
2 无长期记忆  
3 无行动能力

因此需要扩展为 Agent。
