
# 融合层次
- **数据级融合**：原始数据合并、优点（高精度）、挑战（数据量大）
- **特征级融合**：提取特征后融合、应用（如目标识别）
- **决策级融合**：独立决策后融合、优点（鲁棒性高）

# 同步与对齐
- **时间同步**：时间戳对齐、插值方法、硬件同步
- **坐标系标定**：手眼标定、外参标定、工具（如MATLAB标定工具箱）
- 常见问题：延迟补偿、坐标系转换

# 工程组合示例
- **IMU + GPS**：互补滤波、卡尔曼滤波融合
- **Camera + Radar**：目标关联、数据融合策略
- **Camera + LiDAR**：点云与图像融合、3D重建

# 实践项目

- 标定一个相机-雷达系统的外参
