# 5 工程实践与学习路径

## 5.1 阶段目标
1. **基础阶段**：掌握凸优化建模与CVXPY，能解小型LP/QP。
2. **融合阶段**：实现一维/二维KF，扩展到EKF/UKF。
3. **进阶阶段**：做IMU+GPS融合实验，理解状态估计全流程。
4. **跟踪阶段**：学习匈牙利算法，跑通SORT等多目标跟踪demo。
5. **综合阶段**：在仿真或ROS环境下实现多传感器融合 + 多目标跟踪。
