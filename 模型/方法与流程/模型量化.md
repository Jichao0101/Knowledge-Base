# 1 PTQ（训练后量化）

## 1.1 目的与适用范围

PTQ（训练后量化）的目标是：  

- 不重新训练模型
    
- 用低比特逼近浮点模型数值行为

PTQ 适用于：
- 已完成训练、权重冻结的模型
- 对推理性能/资源有约束的部署场景
- 不具备或不希望引入量化感知训练（QAT）的情况

## 1.3 PTQ 的核心任务

PTQ 并不直接优化 accuracy，其核心任务包括：

1. **估计 activation / weight 的统计分布**
2. **决定量化参数（scale / zero-point / clipping）**
3. **最小化量化引入的数值误差**

---

## 1.4 Calibration

### 1.4.1 Calibration 的目的

Calibration 的目标不是评估模型性能，而是：

- 测量各量化点张量的数值分布
- 为量化建立合理的数值映射关系
- 减少浮点 → 低比特的近似误差

Calibration 不需要标签（label-free）。

---

### 1.4.2 校准对象

#### 1.4.2.1 Weight（静态）

- 权重在训练后固定
- 可直接统计：
  - min / max
  - 直方图
- 常用对称量化
- 通常支持 per-channel 量化（尤其卷积）

#### 1.4.2.2 Activation（动态）

- 依赖输入数据
- 需通过 calibration set 跑一遍模型收集统计量
- 常见统计：
  - min / max
  - percentile
  - histogram（用于 KL / MSE）

---

### 1.4.3 Calibration Set

Calibration set 的要求：
- 来自训练集或与训练集同分布
- 覆盖典型输入场景
- 数量不必很大（几十到几百即可）

Calibration set 的职责：
- 估计数值分布
- 确定量化参数

**Calibration set 不用于评估泛化性能。**

---

## 1.5 量化参数的确定

### 1.5.1 线性量化模型

典型线性量化公式：

- Quantize：

$q = \mathrm{clip}(\mathrm{round}(x/s) + z)$


- Dequantize：

$\hat{x} = s \cdot (q - z)$


其中：
- `s`：scale（量化步长）
- `z`：zero-point（零点）
- clip：截断到整数表示范围

---

### 1.5.2 对称 vs 非对称量化

- **对称量化**
  - zero-point = 0
  - 常用于 weight
  - 硬件友好

- **非对称量化**
  - zero-point ≠ 0
  - 更好覆盖偏移分布
  - 常用于 activation

---

### 1.5.3 Clipping 策略

直接使用 min/max 易受 outlier 影响，常见替代策略：

- Percentile（如 99.9%）
- KL-divergence
- MSE 最小化

目标是在：
- 舍入误差（scale 过大）
- 截断误差（clip 过多）

之间取得平衡。

---

### 1.5.4 Per-Tensor vs Per-Channel

- **Per-tensor**
  - 实现简单
  - 精度相对较差

- **Per-channel**
  - 每个通道独立 scale
  - 对卷积权重显著提升精度
  - 依赖硬件/引擎支持

---

## 1.6 PTQ 后的评估

### 1.6.1 Calibration Set（数值近似验证）

关注：
- 量化前后激活/权重误差
- 中间层数值偏差
- 是否出现异常饱和或截断

不关注最终 accuracy。

---

### 1.6.2 Test Set（泛化验证）

Test set 用于回答：

> **量化后的模型在未见数据上是否保持决策一致性？**

关注指标包括：
- Accuracy / Precision / Recall
- 与 FP32 模型的输出一致性
- 类别偏移或系统性误差

Calibration set 无法替代 Test set。

---

## 1.7 常见问题与风险点

- 输入前处理不等效（最常见）
- Calibration 数据分布不代表线上
- Resize / 颜色空间实现不一致
- scale 被 outlier 拉大
- 激活量化策略过于粗糙
- 输出层或敏感层量化误差放大

---

## 1.8 总结

PTQ 的本质是：

> **在保证输入等效的前提下，  
> 用有限比特尽可能逼近浮点模型的数值行为。**
